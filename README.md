# LampShop Frontend (React)

Frontend-часть интернет-магазина на React + React Router DOM.

## Что реализовано

- Главная страница (`/`)
- Каталог (`/catalog`)
- Карточка товара (`/product/:productId`)
- Корзина (`/cart`)
- Оформление заказа (`/checkout`)
- Подтверждение заказа (`/checkout/success`)
- Страницы `О нас` и `Контакты`
- Адаптивная верстка (от 320px)
- Mock-данные без подключения backend
